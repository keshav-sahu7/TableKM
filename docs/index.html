Quick Help

  • Any integer is considered int32 e.g. 10 is int32
  • Any integer followed by `l` or `L` is considered int64 e.g 10L is int64
  • Any floating point number followed by `f` is considered float32 e.g. 10.0f is float32
  • Any floating point number is considered float64 e.g. 10.0 is float64
  • Any text under doubly inverted comma such as "Hello World!" is considered string
  • True and False are only boolean literal value
