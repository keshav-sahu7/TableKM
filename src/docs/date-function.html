This is a list of available date and date_time functions.


  • int32 day( date value)
    int32 day( date_time value)

    Returns day part of the value.



  • int32 month( date value)
    int32 month( date_time value)

    Returns month part of the value.



  • int32 year( date value)
    int32 year( date_time value)

    Returns year part of the value.



  • int32 hour( date_time value)

    Returns hour part of the date_time value.



  • int32 minute( date_time value)

    Returns minute part of the date_time value.



  • int32 second( date_time value)

    Returns second part of the date_time value.



  • boolean isLeapYear( int32 year)

    Returns whether year is leap year or not.
