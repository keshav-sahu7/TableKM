This is a list of available logical functions.


  • boolean NOT( boolean value)

    Returns opposite of value. If value is True then False is returned, if value is False then True is returned.



  • boolean AND( boolean value1, boolean value2)

    Returns True if both value1 and value2 are True, False otherwise.



  • boolean OR( boolean value1, boolean value2)

    Returns True if at least one of value1 and value2 is True, False otherwise.



  • int32 IF( boolean condition, int32 value1, int32 value2)
    int64 IF( boolean condition, int64 value1, int64 value2)
    float32 IF( boolean condition, float32 value1, float32 value2)
    float64 IF( boolean condition, float64 value1, float64 value2)
    string IF( boolean condition, string value1, string value2)
    boolean IF( boolean condition, boolean value1, boolean value2)
    date IF( boolean condition, date value1, date value2)
    date_time IF( boolean condition, date_time value1, date_time value2)

    Returns value1 if condition is True, value2 otherwise. Type of both values must be same. It is a typical if statement.
