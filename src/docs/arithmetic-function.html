This is the list of available arithmetic functions.


  • int32 add ( int32 value1 , int32 value2 )
    int64 add ( int64 value1 , int64 value2 )
    float32 add ( float32 value1 , float32 value2 )
    float64 add ( float64 value1 , float64 value2 )
    string add ( string value1, string value2)

    Returns sum of value1 and value2 . Return type is same as data type of value1 and value2.



  • int32 subtract ( int32 value1 , int32 value2 )
    int64 subtract ( int64 value1 , int64 value2 )
    float32 subtract ( float32 value1 , float32 value2 )
    float64 subtract ( float64 value1 , float64 value2 )

    Returns difference of value1 and value2. Return type is same as data type of value1 and value2. sub is a shorthand for this function.



  • int32 multiply ( int32 value1 , int32 value2 )
    int64 multiply ( int64 value1 , int64 value2 )
    float32 multiply ( float32 value1 , float32 value2 )
    float64 multiply ( float64 value1 , float64 value2 )

    Multiplies value1 and value2 and returns the result. Return type is same as data type of value1 and value2. mul is a shorthand for this function.



  • float32 divide ( int32 value1 , int32 value2 )
    float64 divide ( int64 value1 , int64 value2 )
    float32 divide ( float32 value1 , float32 value2 )
    float64 divide ( float64 value1 , float64 value2 )

    Divides value1 by value2 and returns the result. Return type is float32 in case of int32 and float32 , but float64 in case int64 and float64 . div is a shorthand for this function.



  • int32 intDiv ( int32 value1 , int32 value2 )
    int64 intDiv ( int64 value1 , int64 value2 )

    Return integer division result of value1 by value2. Return type is same as data type of value1 and value2. For example intDiv(7,4) = 1, intDiv(8,4) = 2



  • int32 mod ( int32 value1 , int32 value2 )
    int64 mod ( int64 value1 , int64 value2 )
    float32 mod ( float32 value1 , float32 value2 )
    float64 mod ( float64 value1 , float64 value2 )

    Return mod of value1 and value2. Return type is same as data type of value1 and value2. For example mod(11,2) = 1



  • float32 sqrt ( int32 value)
    float64 sqrt ( int64 value)
    float32 sqrt ( float32 value)
    float64 sqrt ( float64 value)

    Returns square root of value. Return type is float32 in case of int32 and float32 , but float64 in case int64 and float64 .



  • int32 pow ( int32 value1 , int32 value2 )
    int64 pow ( int64 value1 , int64 value2 )
    float32 pow ( float32 value1 , float32 value2 )
    float64 pow ( float64 value1 , float64 value2 )

    Returns value1 to the value2 power. Return type is same as data type of value1 and value2.



  • int32 abs ( int32 value)
    int64 abs ( int64 value)
    float32 abs ( float32 value)
    float64 abs ( float64 value)

    Returns absolute value of value. Return type is same as data type of value. For example abs(10) = 10, abs(-10) = 10



  • float32 floor ( float32 value)
    float64 floor ( float64 value)

    Returns largest integer <= value. Return type is same as data type of value. For example floor(10.1) = 10, floor(10.9) = 10, floor(10) = 10



  • float32 ceil ( float32 value)
    float64 ceil ( float64 value)

    Returns smallest integer >= value. Return type is same as data type of value. For example ceil(10.1) = 11, ceil(10.9) = 11, ceil(10) = 10



  • boolean isInRange ( int32 value, int32 start , int32 end )
    boolean isInRange ( int64 value, int64 start , int64 end )
    boolean isInRange ( float32 value, float32 start , float32 end )
    boolean isInRange ( float64 value, float64 start , float64 end )
    boolean isInRange ( string value, string start , string end )
    boolean isInRange ( date value, date start , date end )
    boolean isInRange ( date_time value, date_time start , date_time end )

    Finds if value lies in inclusive range [ start , end ], that means if start <= value <= end . Returns True if it is in range, False otherwise.
