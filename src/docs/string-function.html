This is the list of available string functions.


  • int32 length( string value)

    Returns length of string value, it returns length of total bytes so only applicable on Latin chars. For example, length("ABCD") = 4.



  • int32 concatenate( string value1, string value2)

    Returns value1 + value2. For example, concatenate("Apple ","Banana") = "Apple Banana"



  • string lowerCase( string value)
    string toLower( string value)

    Returns lower case form of the value. For example, toLower("Billy Jack") = "billy jack".



  • string upperCase( string value)
    string toUpper( string value)

    Returns upper case form of the value. For example, toUpper("Billy Jack") = "BILLY JACK".



  • int32 countChar( string source, string char)

    Counts the number of appearance of char's first byte in source. If char is empty then 0 is returned. If it contains multiple chars, only first char will be searched. For example, countChar("hello", "l except l other chars will be ignored") = 2.



  • boolean contains( string source, string sub_str)

    Returns True if sub_str is a sub string in source, False otherwise. For example, contains("Hello World!", "ello") = True



  • boolean containsAnyOf( string source, string char_group)

    Returns True if any char from char_group exists in source. For example, containsAnyOf("A sentense", "aeiou") = True
