This is the list of available type conversion functions.


  • int32 toInt32( int64 value)
    int32 toInt32( float32 value)
    int32 toInt32( float64 value)
    int32 toInt32( boolean value)

    Converts value of other numeric type to int32 . Be aware this may cause overflow or underflow. In case of boolean , True is converted to 1 and False is converted to 0.



  • int32 toInt32( string value, int32 alternative = 0)
    int32 toInt32B( string value, int32 base, int32 alternative = 0)

    Converts string value to int32 , if value can not be converted to int32 then alternative is returned. Default is 0 if ommitted. base is base of integer. In first function it is 10, in second function, base will be provided by user. 2,8,10,16 are popular base for numbers. For example, toInt32B("A",16) = 10, toInt32B("A",10,100) = 100.



  • int32 toInt64( int32 value)
    int32 toInt64( float32 value)
    int32 toInt64( float64 value)
    int32 toInt64( boolean value)

    Converts value of other numeric type to int64 . Be aware this may cause overflow or underflow in case if value is float32 or float64 . In case of boolean , True is converted to 1 and False is converted to 0.



  • int64 toInt64( string value, int64 alternative = 0)
    int64 toInt64B( string value, int32 base, int64 alternative = 0)

    Converts string value to int64 , if value can not be converted to int64 then alternative is returned. Default is 0 if ommitted. base is base of integer. In first function it is 10, in second function, base will be provided by user. 2,8,10,16 are popular base for numbers.



  • float32 toFloat32( int32 value)
    float32 toFloat32( int64 value)
    float32 toFloat32( float64 value)
    float32 toFloat32( string value, float32 alternative = 0)

    Converts value of other numeric type to float32 . In case of string value to float32 , alternative is returned if it fails to convert value to float32 , default alternative is 0.



  • float64 toFloat64( int32 value)
    float64 toFloat64( int64 value)
    float64 toFloat64( float32 value)
    float64 toFloat64( string value, float32 alternative = 0)

    Converts value of other numeric type to float64 . In case of string value to float64 , alternative is returned if it fails to convert value to float64 , default alternative is 0.



  • string toString( int32 value)
    string toString( int64 value)
    string toString( float32 value)
    string toString( float64 value)
    string toString( boolean value)

    Converts value of any numeric type to string . 10 is used as base for integers. In case of boolean value, True is converted to "True" and False is converted to "False"



  • boolean toBoolean( int32 value)
    boolean toBoolean( int64 value)

    If value is nonzero then True is returned, False otherwise.



  • date toDate( int32 day, int32 month, int32 year)

    Date is constructed by given day, month and year. Note : it doesn't check for validity of the date . It is user's responsiblity.



  • date_time toDateTime( int32 day, int32 month, int32 year, int32 hour, int32 minute, int32 second)
    date_time toDateTime( date date_value, int32 hour, int32 minute, int32 second)

    Date is constructed by given date and time components. Note : it doesn't check for validity of the date . It is user's responsiblity.
